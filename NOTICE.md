# Data Source & Attribution

This integration retrieves public transport timetable and departure data at runtime from the **transport.opendata.ch** API, a free service operated by [Opendata.ch](https://opendata.ch/) that is backed by the timetable of **search.ch**.

The API requires no account or key. Please keep usage reasonable — it is a community service with rate limits.

**Attribution:** *Data: transport.opendata.ch (search.ch)*

Every entity this integration creates sets Home Assistant's `attribution` attribute accordingly, surfaced in the entity's "More Info" dialog. If you build dashboards or republish this data, please keep that attribution visible.

This integration is unofficial and not affiliated with, endorsed by, or supported by Opendata.ch, search.ch, or any transport operator.
